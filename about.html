The DC Code: About

This website is run as a volunteer effort by dccode.org: it's an unofficial presentation of the first free copy of the Washington, DC code.

As such, we need to be clear that there is no warranty or guarantee, and that the most up-to-date changes that have happened in the official copy.

The website is a free and open source project that you can contribute to with bug reports, improvements, and more.
